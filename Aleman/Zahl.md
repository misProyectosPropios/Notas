## Ordinalzahlen: Datum (Ordinal Numbers for Dates)

| **Number** | **German Ordinal (English)** | **Notes on Formation**                      |
| ---------- | ---------------------------- | ------------------------------------------- |
| **1.**     | **ersten** (first)           | Irregular ending.                           |
| **2.**     | **zweiten** (second)         | Irregular ending.                           |
| **3.**     | **dritten** (third)          | Irregular ending.                           |
| **4.**     | **vierten** (fourth)         | Regular ending for 4-19.                    |
| **5.**     | **fünften** (fifth)          | Regular ending for 4-19.                    |
| **6.**     | **sechsten** (sixth)         | **sechs** (six) loses the 's'.              |
| **7.**     | **siebten** (seventh)        | **sieben** (seven) loses the 'en'.          |
| **8.**     | **achten** (eighth)          | **acht** (eight) already ends in 't'.       |
| **9.**     | **neunten** (ninth)          | Regular ending for 4-19.                    |
| **10.**    | **zehnten** (tenth)          | Regular ending for 4-19.                    |
| **20.**    | **zwanzigsten** (twentieth)  | Uses the suffix **-sten** for 20 and above. |
| **30.**    | **dreißigsten** (thirtieth)  | Uses the suffix **-sten** for 20 and above. |
